# Paper sample images

This directory contains five small input/enhanced sample pairs extracted from the qualitative figures in the APCF-Net manuscript. They are included only for repository verification, visual inspection, and README illustration.

These files are **not a redistribution of the full UIEB, UCCS, or NYU-v2 datasets**. For full training and evaluation, download the datasets from their original providers and place the raw images under `data/raw/`.

Directory contents:

- `input/`: sample raw/degraded underwater images used for quick inference checks.
- `enhanced/`: corresponding enhanced images shown in the manuscript figures.
- `sample_pairs.csv`: mapping between each input sample and its manuscript enhanced result.
