# Saved models

Training saves checkpoints here.

Default output path:

```text
saved_models/apcfnet/best.pth
saved_models/apcfnet/last.pth
saved_models/apcfnet/config_snapshot.yaml
```

The public repository does not include the final trained checkpoint by default. After training on the full dataset, place the final checkpoint in `saved_models/apcfnet/best.pth` or archive it through the same Zenodo release as the code if file size permits.
