# Results directory

Generated outputs are written here.

Expected files after full inference/evaluation:

```text
results/enhanced/                         # enhanced test images generated by APCF-Net
results/metrics/no_reference_metrics.csv  # no-reference metrics for raw/non-paired outputs
results/metrics/reference_metrics.csv     # optional PSNR/SSIM when references are available
results/figures/                          # qualitative figures and previews
```

This repository also includes:

```text
results/paper_samples/enhanced/           # five enhanced examples extracted from manuscript figures
results/figures/paper_sample_pairs.png    # input/enhanced sample grid
results/metrics/paper_sample_no_reference_metrics.csv
```

The paper-sample metrics are provided only to verify that the metric script runs on included images. Full manuscript tables should be generated from the complete dataset splits used in the paper.
